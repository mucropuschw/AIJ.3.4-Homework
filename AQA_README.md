# Leap Year Tests

## Description
Autotests for validating leap year calculation logic.

## Covered scenarios
- divisible by 400
- divisible by 100 but not 400
- divisible by 4
- non-leap years
- invalid years
- boundary values

## Requirements
- Python 3.11+
- pytest

## Run tests

Using pytest:

```bash
pytest -v
python -m unittest test_leap_year.py
